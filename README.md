### 어떻게 동작할까?

로딩이 된다면, 기본 state를 초기화하고
그 state를 기반으로 화면을 보여준다. 

- 지갑에 돈을 얼마나 넣을것인가
- 랜덤으로 부여 or 가중치 랜덤 
- 최대값이나 최소값을 부여할 것인가 
- 돈 반환의 경우, 100원짜리 10개 넣고 반환하면 1000원주나? 
아니면 들어온 화폐에 맞게 다시 반환해주나? 


- 상품의 경우에도 랜덤으로 갯수를 정한다. 최소 최대 갯수. 
- 가격은 우리가 설정한다. 

- 5초동안 입력 안 하면 반환하는 논리는 디바운스를 참고
